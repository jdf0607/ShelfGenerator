# Shelf Generator

## Descripción
Proyecto de generación de estanterías para la asignatura de Proyectos de Programación. Te permite establecer productos 
con relación entre sí, mediante tags o valores de similitud, y genera una estantería con el mejor orden posible.

## Como comenzar

### Prerrequisitos
- Java 22.*
- Gradle

### Installation
1. Clonar el repositorio:
    ```sh
    git clone https://github.com/MeLlamanMeca/your-repo-name.git
    ```
2. Navega al directorio del proyecto:
    ```sh
    cd your-repo-name
    ```
3. Construir el proyecto:
    ```sh
    ./gradlew build
    ```
4. Ejecutar el proyecto:
    ```sh
   java -jar build/libs/ShelfGenerator-1.0-all.jar
    ```
